# Github Navigator

Digite um nome de usuário do GitHub e navegue pelos seus repositórios, seguidores e pessoas seguidas. 

Projeto criado em REACT com design responsivo.

## Demonstração

Veja demonstração em:

* React
* react-router-dom
* Material UI
